# ETS-Vibes CLI

Command-line save editor for Euro Truck Simulator 2 and American Truck Simulator.

## Installation

```bash
cd apps/cli
python -m venv .venv
source .venv/bin/activate
pip install -e .
```

## Usage

```bash
ets-vibes list
ets-vibes edit 1 --money 50000000
ets-vibes quick 100000000
```

See main [README](../../README.md) for more details.
